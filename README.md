# Курс по Unreal Engine 5 (RU)

Это репозиторий с материалами к моему бесплатному курсу по Unreal Engine 5 в сопровождении [YouTube](https://www.youtube.com/@mixseil ) и [ВКвидео.](https://vkvideo.ru/@club236677349)

Здесь лежат:
- конспекты по модулям и занятиям;
- формулировки домашних заданий;
- дополнительные материалы.

## Как устроен курс

Курс построен по модулям:

- Модуль 0 — Введение и установка UE5
- Модуль 1 — Основы работы в редакторе
- Модуль 2 — Дизайн уровней
- Модуль 3 — Аудио
- Модуль 4 — Материалы
- Модуль 5 — Blueprint: основы программирования
- Модуль 6 — ООП и архитектура
- Модуль 7 — Анимация
- Модуль 8 — UI и виджеты 
- Модуль 9 — PCG и процедурная генерация
- Модуль 10 — Niagara и VFX
- Модуль 11 — Разрушения и симуляции
- Модуль 12 — Синематики в Sequencer
- Модуль 13 — Motion Design
- Модуль 14 — Мультиплеер
- Модуль 15 — Полезные инструменты
- ?Модуль 16 — Введение в C++ в UE
- ?Модуль 17 — Всё для создания 2D игры
- ?Модуль 18 — Всё для создания VR игры

Подробный план по занятиям — в каталоге [`LESSONS`](./LESSONS).

## Как пользоваться репозиторием

1. Видео по курсу не тут, они на [YouTube](https://www.youtube.com/@mixseil ) или [ВКвидео.](https://vkvideo.ru/@club236677349)

2. В каталоге [`LESSONS`](./LESSONS) найдите нужный модуль и занятие.  
   В начале каждого `.md` файла есть ссылка на соответствующее видео.

3. Домашние задания к занятиям лежат в [`HOMEWORKS`](./HOMEWORKS),  
   а примеры проектов — в [`PROJECTS`](./PROJECTS).

# Как установить Unreal Engine 5
## Способы
1. Через официальный сайт  
2. Через GitHub (исходники)  
3. Через лаунчер Epic Games  

## По шагам: способ через лаунчер

### 1. Установите Epic Games Launcher
- Скачайте установщик с официального сайта Epic Games  
- Запустите его и завершите установку  

### 2. Войдите в свой аккаунт
- Запустите лаунчер  
- Войдите под своей учётной записью Epic Games (или создайте её)  

> **Важно:** если у вас нет аккаунта, создавайте именно аккаунт Epic Games, а не входите через Google — это поможет избежать проблем в будущем.

### 3. Перейдите в раздел Unreal Engine
- В левом меню лаунчера нажмите вкладку **"Unreal Engine"**  

### 4. Перейдите во вкладку "Библиотека"

### 5. Нажмите "+"
- В верхней части раздела будет кнопка **"+ / Установить" (Install)**  

### 6. Выберите версию и настройки
- Выберите нужную версию (рекомендуется последняя стабильная, на момент записи — **5.6.2**)  
- Укажите путь установки (убедитесь, что есть ~20–30 ГБ свободного места)  
- Выберите компоненты:
  - Для начала достаточно ядра (Unreal Engine)  
  - Остальное можно добавить позже  
  - Для курса нужны **первые 3 галки**  

### 7. Запустите установку
- Нажмите **"Установить" (Install)**  
- Дождитесь завершения загрузки и распаковки  

### 8. Запустите UE5
- После установки появится кнопка **"Запуск" (Launch)**  
- Также ярлык появится на рабочем столе

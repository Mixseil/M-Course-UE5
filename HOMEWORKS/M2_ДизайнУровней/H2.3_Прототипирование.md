# Домашка 2.3: Прототипирование - Средневековое поселение

## Задача

Создать несколько набросков домиков в средневековом стиле, используя примитивы (кубы, цилиндры, конусы) с инструментами modeling tools и/или с BSP геометрией. Один из домов должен быть двухэтажным с лестницей. 

Когда придумывал это задание думал про домики в Ведьмаке, но вы можете придумать что-то интереснее, если есть идеи.

## Что должно быть в результате

- **3-4 домика** разных размеров и форм в средневековой планировке
- **Один дом** - двухэтажный с внутренней лестницей
- Домики должны быть **расположены рядом** на плоской площадке
- Использованы только **примитивные формы** (Cube, Sphere, Cone, Cylinder)
- Базовые **материалы** для визуализации (дерево, камень)

## Этапы выполнения

### Этап 1: Подготовьте уровень

### Этап 2: Создание простых домиков
**Несколько простых**
   - Стены: Можно использовать как отедльные блоки для каждой стены так и один объект полый внутри. 
   - Крыша: Cone с вершиной наверху (типичный средневековый треугольный скат)
   - Дверь: прямоугольный Cube, врезанный в стену (или вырезанный отверстие)
   - Окна: маленькие Cube'ы в стене создающие проёмы. А можно и целиндры
   - Можете добавить деталей: дымоход (Cylinder), ставни у окон
   - Используйте разные пропорции (квадратный или широкий)
   - Экспериментируйте с формой крыши

**Двухэтажный дом с лестницей**
   - те же принципы, можете поиграться с размерами
   - можете создать несколько двухэтажных, если не жалко время и нравится процесс
   - Лестница: при помощи отдельного объекта лестницы

***Рынок**
   - задание со звёдздочкой, можете накидать немного торговых полаток и сформировать небольшую торговую площадку


### Этап 3: Размещение и композиция
1. Расположите все домики на плоскости в виде небольшого поселения
2. Оставляйте 1-2 метра между домиками для прохода
3. Один дом может быть немного дальше или ближе для визуального разнообразия
4. Центр сцены - двухэтажный дом, вокруг него 3 одноэтажных
5. *Можете добавть паре домов ипровизированные дворики
6. *Можете на небольшой площадке между домами поставить цилиндр а на дним параллепипед (типа статуя или фонтан). Можете сделать что сложнее если есть идеи, желанные к реализации.

## Ещё задание со звёздочкой ⭐

- Добавьте в дома стол, стулья, шкаф.

**Используйте инструмент из Modeling Tools для создания детали:**
- В боксели режиме (Modeling Mode → Boolean Operations) создайте **сложную форму дверного проёма** - не просто прямоугольник, а арка (используйте Boolean Subtract для вырезания). Можете использовать фаску для сглаживания проёма.
- При выполнении других заданий со звёздочкой у вас на уровне будет фонтан со статуей, можете воспользоваться инструментом скручивания чтобы статуя не была просто параллепипидом, а выглядела более интересно.

## Советы

- Не усложняйте слишком - это прототипирование, цель показать композицию и размеры
- Используйте разыне размеры для разнообразия (большой дом, маленький домик, средний)
- Средневековые дома обычно имеют скатную крышу - используйте конусы

4. Если выполняли задание со звёздочкой - объясните, какой инструмент использовали и почему

# Домашка 2.4: Ландшафт - Холмистое поселение

## Задача

Создать холмистый ландшафт с использованием карты высот, на котором разместить ранее созданное поселение домиков в прошлой домашке. Добавить дорожки между домами, варьировать высоты расположения домов. Творчески отредактировать ландшафт после импорта.

## Что должно быть в результате

- **Ландшафт** создан из карты высот (импортирован, не создан вручную)
- **Холмистая местность** с перепадами высот
- **Поселение** (домики из предыдущей домашки) расположено на ландшафте:
  - Домики на разных высотах (один повыше, один пониже)
  - Друг к другу ниже/выше по склону
- **Дорожки** между домами (созданы из Spline или плоских платформ)
- **Творческие редакты** ландшафта (сглаживание, эрозия, добавление деталей)

## Этапы выполнения

### Этап 1: Поиск и подготовка карты высот
1. Найдите подходящую карту высот для холмистой местности:
   - Сайты: FreePBR, OpenGameArt, CGTextures, ArtStation
   - Ищите карту размером **1024x1024** или **2048x2048**
   - Подойдут файлы в формате PNG, TGA, RAW
   - Убедитесь, что это **16-битное изображение**
   
2. Скачайте и сохраните карту в папку проекта `Content/Textures/Heightmaps/`

3. (Опционально) Отредактируйте карту в графическом редакторе (Photoshop, figma):
   - Увеличьте контрастность для более выразительных холмов
   - Сгладьте слишком резкие переходы, если нужно
   - Добавьте свои штрихи для уникальности

### Этап 2: Импорт ландшафта в UE5
1. Откройте новый или существующий уровень
2. Перейдите в режим **Landscape** (слева в панели режимов)
3. Нажмите **Import Heightmap**
4. Выберите вашу карту высот
5. Установите параметры:
   - **Heightmap Size:** выберите размер, совпадающий с вашей картой (1024x1024, 2048x2048)
   - **Z Scale:** установите 100-150 для выразительных холмов (экспериментируйте)
   - **X/Y Scale:** оставьте 1.0 или увеличьте для более крупного ландшафта
6. Нажмите **Import** и дождитесь создания ландшафта

### Этап 3: Редактирование ландшафта
1. Остаётесь в режиме **Landscape**
2. Используйте инструменты редактирования:
   - **Sculpt** - сформируйте плоские площадки под домики
   - **Smooth** - сгладьте резкие переходы, сделайте холмы натуральнее
   - **Flatten** - создайте ровные участки для домов
   - **Erosion** - добавьте природности, имитируя эрозию на склонах

3. Сделайте творческие изменения:
   - Добавьте небольшую впадину или оврагчик
   - Создайте видимые пути (уходящие вниз дорожки)
   - Варьируйте высоты для интересной композиции

### Этап 4: Размещение поселения на ландшафте
1. Импортируйте или переместите ваши домики из предыдущей домашки на ландшафт
2. **Расположите домики на разных высотах:**
   - Один домик повыше (на вершине холма)
   - Один домик пониже (в низине или на склоне)
   - Остальные домики на средних высотах
   Это создаст интересную композицию поселения

### Этап 5: Создание дорожек
1. **Вариант 1: Spline-based дорожки** (рекомендуется)
   - Create → Spline Actor
   - Проложите путь между домиками
   - Используйте инструмент Spline для создания плавных переходов
   - Создайте mesh дорожки из простого куба (примитива), вытянутого по spline
   - Материал дорожки: серый камень или земля, можете импортировать из Fab материал, но это опционально

2. **Вариант 2: Платформы**
   - Создайте плоские платформы (тонкие Cube'ы) между домиками
   - Следуйте рельефу, поднимаясь и опускаясь со склонов

3. Дорожки должны:
   - Соединять домики
   - Следовать рельефу ландшафта
   - Иметь видимую отличие от земли (другой материал, немного приподняты)

### Этап 6: Финальная композиция
1. Снимите режим Landscape (выберите другой режим редактирования)
2. Посмотрите на сцену со всех сторон
3. Отрегулируйте положение домиков и дорожек для лучшей композиции
 Можете вообще добавить воду (простой plane синего цвета) в низине, если хотите

## Задания со звёздочкой ⭐ без описания, что нужно сделать по видео

 1. Сделайте материал ландшафта с видимостью маской
 2. Сделайте у какого-нибудь домика подвал, чтоб туда можно было пройти через подходяшее отверстие в ландшафте.


## Советы

- **Карта высот** - это вашей будущего ландшафта основа, всё остальное редактируется в UE5
- **Z Scale** экспериментируйте - не бойтесь менять значение для лучшего результата
- **Спланируйте** расположение домиков до размещения - продумайте, где будут дорожки
- Дорожки должны быть видимы издалека - не делайте их слишком узкими
- Используйте **Flatten** для создания плоских участков под домики
- Если ландшафт выглядит неестественно - используйте **Smooth** и **Erosion**
- Подвал - это сложное задание, не делайте слишком большим, сфокусируйтесь на качестве одного дома

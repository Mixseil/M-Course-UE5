# Домашка 2.7: Атмосфера - Полная атмосферная ночная сцена с облаками и туманом

## Задача

Завершить создание атмосферной ночной сцены средневекового поселения в лесу на холме, используя облака, туман, пост-обработку и тщательную организацию сцены. 

**Это домашка продолжает сцену из прошлых** (ночная сцена с факелами и луной). Здесь мы добавляем атмосферные эффекты, которые создадут магическое, грустное и немного опасное ночное настроение на улице, и уют с теплом внутри домов.

## Что должно быть в результате

- **Облака новые (Volumetric Cloud)**: красивые объёмные облака, через которые видна луна, измененный цвет (холодные тона ночи)
- **Облака старые (SkySphere)**: слегка видимые звёзды, облака тёплого цвета поверх (слои облаков для глубины)
- **Туман глобальный (Exponential Height Fog)**: создаёт глубину сцены, особенно в низинах, холодный голубой оттенок
- **Туман локальный (Local Fog Volume)**: в лесу вокруг деревьев, у подножия холма для эффекта ночной "гущи"
- **God Rays эффект**: лучи света от луны сквозь облака и туман (если выполнили бонус)
- **Пост-процесс на всю карту**: единая экспозиция, цветокоррекция для ночной атмосферы
- **Пост-процесс в домах**: теплое свечение внутри дома (желтые/оранжевые тона), небольшой контраст настроения между холодной улицей и теплым домом
- **Физически точный свет**: луна светит реалистично по своей интенсивности, свет факелов взаимодействует с туманом, тени правильные, баланс света соблюдён
- **Порядок в организации**: все экторы рассортированы по папкам (Environment, Fog, Lights, Meshes и т.д.), использованы слои

## Важно

Все параметры подбирайте **самостоятельно** исходя из:
- Реального мира
- Своего видения (что вам нравится визуально)
- Характера сцены: **ночь — грустная и опасная на улице**, **дома — тёплые и уютные**
- Физических реалий: луна светит холодно и слабо, факелы светят тепло, дома светятся жёлтым
- Производительности: постоянно следите за FPS, не добавляйте слишком много объёмного тумана

## Этапы выполнения

### Этап 1: Подготовка (копирование лекции 2.6)

1. Убедитесь, что у вас есть рабочая сцена из **H2.6**:
   - Directional Light для луны
   - Факелы на столбах и у дверей домов
   - Свет в окнах домов
   - SkyLight и SkyAtmosphere

### Этап 2: Добавление облаков

#### 2.1 Старые облака (SkySphere)

1. **Добавьте BP_SkySphere:**
   - Content Browser → Settings (шестерёнка) → Show Engine Content
   - Найдите `BP_SkySphere` в Engine Content
   - Перетащите на уровень (или используйте Place Actor)

2. **Настройте SkySphere для ночи:**
   - **Direction Light Actor**: выберите вашу луну (Directional Light)
   - **Refresh Material**: нажмите кнопку
   - **Colors Determined By Sun Position**: отключите (галочку снимите), чтобы настраивать вручную
   - **Override Settings** → **Zenith Color**: тёмный фиолетовый/чёрный (ночное небо)
   - **Override Settings** → **Horizon Color**: тёмный голубой с фиолетовым оттенком (плавный переход к линии горизонта)
   - **Cloud Color**: холодный голубой (почти белый, холодного оттенка) — облака ночью светлые от луны
   - **Cloud Opacity**: 0.3-0.5 (не очень плотные, чтобы видна была луна)
   - **Cloud Speed**: 0.02-0.05 (медленное движение облаков)
   - **Stars Brightness**: 0.5-0.8 (видимые звёзды ночью, но не перепалощающие облака)

3. **Скопируйте BP_SkySphere в вашу папку проекта** (если хотите внутренне редактировать):
   - ПКМ на BP_SkySphere → Duplicate → выберите папку вашего проекта
   - Удалите старый из Engine Content с уровня
   - Добавьте новый (скопированный) на уровень

#### 2.2 Новые облака (Volumetric Cloud)

1. **Добавьте Volumetric Cloud:**
   - Place Actors → Volumetric Cloud (или вытащите из Content Browser, если найдёте)

2. **Настройте для ночной сцены:**
   Приведу описение: 
   - Низкая дальность облаков, ибо ночь, да и сфера неба далёкие перекроит
   - Низкая плотность облаков, чтобы видны были звёзды
   - Остальное по вашему виденью реального мира. Некоторые настроки и в этой домашке может стоило заменить или отменить ;)

3. Volumetric Cloud требовательны. Если FPS падает ниже 30, уменьшите покрытие или разрешение облаков.

### Этап 3: Добавление тумана

#### 3.1 Глобальный туман (Exponential Height Fog)

1. **Добавьте или отредактируйте Exponential Height Fog:**
   - Place Actors → Exponential Height Fog

2. **Настройте для ночной сцены:**
   - **Fog Density**: 0.01-0.04 (слабый, но ощутимый туман)
   - **Temp** Выстовите холодную температуру, или слегка холодную (если нет парааметра температуры выставите нужный оттенок)
   - **Directional Inscattering Color**: слегка теплый (луна сквозь туман)
   - **Height Fog Falloff**: 0.2 (туман гуще в низинах)
   - **Volumetric Fog**: ✓ включите (туман будет взаимодействовать с освещением)
   - **Volumetric Fog Scattering Distribution**: 0.5-0.7
   - **Volumetric Fog Ambient Scattering Scale**: 0.3-0.5 (чтобы окружение не было полностью чёрным в тумане)
   - Прочие настройки, которыми вы пигрались и считаете нужными

3. **В Directional Light (луна) включите:**
   - **Light Shaft Occlusion**: ✓
   - **Light Shaft Bloom**: ✓ это всё для год рейсов
   

#### 3.2 Локальный туман (Local Fog Volume)

1. **Добавьте несколько Local Fog Volume:**
   - Place Actors → Local Fog Volume
   - Создайте **минимум 2-3 волюма**:
     - Один в лесу вокруг основной деревни
     - Один на склоне холма
     - Один внутри плотного леса (если есть)

2. **Настройка каждого Local Fog:**
   - **Density**: сделаейте гуще, чем глобальный
   - **Fog Inscattering Color**: холодный голубой (совпадает с глобальным)
   - **Falloff**: 0.1-0.2 (туман размывается к краям волюма)
   - **Ambient Scattering**: 0.2-0.3
   - **Size**: отрегулируйте, чтобы туман покрывал нужную область (можно масштабировать в редакторе)

3. **Подсказка:** используйте Show → Advanced → Show Fog Volumes, чтобы видеть границы волюмов в редакторе.

### Этап 4: Пост-обработка и цветокоррекция

#### 4.0
1.  Откройте карту **Content Examples** (обычно идёт с Engine Content) и зайдите в настройки через **Show Modifier Properties** — там вы найдёте уже настроенные post process volumes, можете посмотреть параметры и вдохновиться. 

> Также можете найти бесплатные готовые ассеты на **Fab** (Sketchfab) и адаптировать их под вашу сцену. Ну или посмотреть, что я ниже понаписал.

#### 4.1 Глобальный Post Process Volume (на всю карту)

1. **Добавьте Post Process Volume:**
   - Place Actors → Post Process Volume

2. **Настройте как глобальный:**
   - **Infinite Extent (Unbound)**: ✓ включите (волюм действует на весь уровень)

3. **Раздел Exposure:**
   - **Exposure Compensation**: 0 (чтобы выставить физически точные параметры, потом можно вернуться к этому значению уменьшив его для ночи или увеличив, но сейчас если тут не 0 то будет только мешать)
   - **Min Brightness**: 0.1 (не допускаем полную чёрноту)
   - **Max Brightness**: 2.0 (луна и факелы должны быть видны, но не выбивать яркость)

4. **Раздел Film:**
   - **Slope**: 0.8-0.88 (мягкий контраст для ночной сцены)
   - **Toe**: 0.3-0.4 (мягкие тёмные области, не резкие чёрные)
   - **Shoulder**: 0.5

5. **Раздел Global (удобно использовать окно Color Grading):**
   - **Saturation**: 0.8-0.9 (ночью цвета менее насыщены)
   - **Contrast**: 0.7-0.8 (небольшой контраст для глубины)
   - **Gamma**: 1.2-1.3 (немного поднимаем, чтобы видеть детали в тенях)

6. **Раздел Temperature (Color Grading):**
   - **Temp**: холодные ночные тона
   - **Tint**: ёгкий фиолетовый оттенок

7. **Раздел Shadows (Color Grading):**
   - **Saturation**: 0.8 (тени менее насыщены)
   - **Contrast**: 0.6
   - **Gamma**: 1.1

8. **Раздел Highlights (Color Grading):**
   - **Saturation**: 1.0-1.1 (луна и факелы остаются яркими)

#### 4.2 Локальный Post Process Volume (внутри дома)

1. **Добавьте второй Post Process Volume:**
   - Place Actors → Post Process Volume
   - Поместите ВНУТРИ одного из домов (или рядом с дверью дома)

2. **Настройте как локальный (тёплый):**
   - **Infinite Extent**: ✗ отключите
   - **Blend Radius**: 200-500 (плавный переход между холодной ночью и теплом дома)
   - **Blend Weight**: 1.0 (чтоб работал в условиях глобального ППВ(постпроцесс волиума))

3. **Раздел Exposure:**
   - **Exposure Compensation**: 0 до 0.2 (внутри немного светлее, но всё равно без особого влияния на наши параметры)
   - **Min Brightness**: 0.3

4. **Раздел Color Grading (Global):**
   - **Saturation**: 1.0-1.1 (цвета живые и тёплые)
   - **Contrast**: 0.8

5. **Раздел Temperature (Color Grading):**
   - **Temp**: 10-20 (ТЁПЛЫЕ тона — жёлтый/оранжевый свет факелов)
   - **Tint**: 5-10 (тёплый оттенок)

6. **Раздел Shadows (Color Grading):**
   - **Saturation**: 0.9
   - **Gamma**: 1.15

7. **Раздел Highlights (Color Grading):**
   - **Saturation**: 1.1-1.2 (тепло подсвеченных участков)

#### 5.2 Нейминг актёров

Убедитесь, что все объекты имеют понятные имена (НЕ "Actor", "Actor1", "Cube", "Sphere"):
- ✓ `DirectionalLight_Moon`
- ✓ `PointLight_Torch_Center_01`
- ✓ `House_WoodCottage_01`
- ✓ `Wall_01`
- ✓ `LocalFogVolume_Forest`
- ✗ Actor, Actor1, Light1, Box, Sphere

#### 5.3 Слои (Layers)

Создайте слои для удобства работы:
- Window → Layers → ПКМ → Create Layer

Примеры слоёв:
- **House 1**: lдом 1
- **Day Atmosphere**: все экторы создающие дневную атмосферу по настройке
- **Night Atmosphere**: соответсвенно

**Как использовать:**
- Выберите актор → детали → Layers → поставьте галочку нужного слоя
- Можно скрывать/показывать целые слои в Layers панели для удобства работы
- Добавить объекты на слой: выделить побъект, ПКМ по слою -> Move selected to layers

### Этап 6: Финальная подстройка физического света и атмосферы

#### 6.1 Проверка физического света

По таблице из конспекта и видео

#### 6.2 Проверка атмосферного эффекта

1. **Включите визуализацию HDR** (для проверки):
   - View Mode → Visualize → HDR (Eye Adaptation)

2. **Проверьте, что:**
   - Луна светит холодно (голубые тени)
   - Факелы светят тепло (оранжевые блики)
   - Контраст между холодной улицей и теплым домом ясен
   - Туман движется (если есть ветер в Exponential Height Fog)
   - God Rays видны (если включили эффект)

3. **Выключите HDR визуализацию:**
   - View Mode → Lit (или другой обычный режим)

3. При необходимости, да и ради эксперемента, меняейте и крутите всё, что вздумается!

#### 6.3 Проверка производительности

1. **Откройте консоль:** Backtick (`)
2. **Введите `stat fps`** — смотрите FPS
3. **Введите `stat unit`** — смотрите Frame Time
4. **Требование:** минимум **45 FPS** на вашем компьютере
5. **Если FPS ниже:**
   - Уменьшите Volumetric Cloud Coverage, или удалите
   - Уменьшите Fog Density
   - Уменьшите количество Local Fog Volumes
   - Отключите Light Shaft эффекты (если ставили)
   - Уменьшите разрешение освещения (Scalability Settings)

2. **Переименуйте все объекты** с понятными именами
3. **Удалите все "temp", "test", "old" объекты** — чистая сцена

5. **Проверьте финальный FPS** и убедитесь, что сцена оптимизирована

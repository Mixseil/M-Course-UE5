# Домашка 2.5: Растительность - Лес и озеленение поселения

## Задача

Заполнить ландшафт растительностью (лес, трава), расположить камни и ветки. Создать атмосферное окружение для средневекового поселения. **Важно:** избегать тяжёлых ассетов, тщательно выбирать модели для оптимизации. 

Помните пределы вашего компьютера и тот факт, что деталезированный лес без огромного колличества ухищрений оптимизации запускается только на мощьных компьютерах для синематиков, не смотря на всю прелесть движка!

## Что должно быть в результате

- **Лес** на склонах и вокруг поселения (деревья различных типов)
- **Трава** на всём ландшафте
- **Камни и валуны** разбросаны в природных местах
- **Ветки и хворост** для создания деталей
- Сцена остаётся **оптимизированной** (у вас она работает хотя бы в 30 FPS)
- **Естественное распределение** - больше деревьев на опушке, травы везде, камней у подножия холмов, без черзмерного наслаивания друг на друга

### Как выбирать ассеты если слабый комп:
1. **Избегайте** сложных ассетов с множеством LOD-уровней (они всё равно нагружают)
2. **Предпочитайте** простые low-poly модели деревьев
3. **Проверяйте** полигон-кол-во (должно быть < 5000 полигонов на одно дерево)
4. **Используйте** несколько экземпляров 1-2 моделей вместо много разных
5. **Читайте отзывы** в Fab, может быть написано про производительность

## Этапы выполнения

### Этап 1: Поиск и добавление ассетов из Fab

1. **Откройте Fab:**
   - В UE5: Tools → Fab или Window → Fab
   - Войдите в свой аккаунт Epic Games (обычно уже авторизованы)

2. **Ищите элементы:**
   - **Деревья**: Ищите "Low poly tree", "Medieval tree", "Forest tree"
     - Выбирайте модели, где указано "Low-poly" или "Game-ready"
     - **Максимум 2-3 типа деревьев** для разнообразия
   
   - **Трава**: "Grass", "Terrain grass texture" или просто текстура травы
     - Достаточно одной хорошей травы
   
   - **Камни/валуны**: "Rock", "Stone", "Boulder", "Medieval stone"
     - Выбирайте 2-3 размера (маленький, средний, большой)
   
   - **Ветки/хворост**: "Branches", "Sticks", "Foliage detail"
     - Простые детали для природного выглядящего рассеивания

3. **Добавляйте в проект.**


### Этап 2: Настройка и импорт в уровень

Если вы выбрали ассеты под реализм, дополнительно проверьте их дитализацию, посмотрите на LOD возможно по средствам движка вы можеет удачно срезать много лишних полигонов.

### Этап 3: Размещение растительности с Foliage Tool

1. **Откройте Foliage Tool:**
   - Режимы редактирования → Foliage Mode

2. **Добавляйте типы растительности в нужное окно элементов**

3. **Настройте параметры для кисти и самметов:**
   - **Brush Size**: 2000-3000 (достаточно крупная кисть)
   - **Density**: 10-20 (не слишком густо, чтобы не лагало)
   - **Random Rotation**: Yaw, Pith, Roll
   - **Random Scale**: включите (0.9-1.1)
   - **Random Z**
   - **Align to Normal**: включите (деревья будут перпендикулярны склону)

  все эти настройки помогают добиться естественности вашему лесу

4. **Рисуйте лес** с учётом созданного в прошлых ДЗ поселения

5. **Добавляйте траву** она может сильно нагружать, её не должно быть избыточно. Можете воспользоваться отключением галочки `Cast Shadow`

6. **Разместите камни**. Старайтесь размещать естественно, там где они могли бы быть.

7. **Добавьте ветки/хворост.**

### Этап 4: Проверка оптимизации

1. **Запустите сцену** (Play)
2. **Проверяйте FPS:**
   - Должно быть > 30 FPS (минимум)
   - На хорошем ПК должно быть 60+ FPS
   - Если меньше - удалите часть растительности, выключите тени. Если уже некуда то ждите следующих занятий)

3. **Визуально проверьте эстетику по вкусу:**
   - Сцена выглядит живой?
   - Травы достаточно, но не слишком много?
   - Лес выглядит естественно?

### *Этап 5: +

Можете добавить деталей
   - Небольшие кусты на дорожках (если есть ассет)
   - Заборчики вдоль дороги (можно сделать ипровезированные по средством инструментов движка)

### *Этап 6: ++
Можете попробовать включить на деревьев Nanite и замерить проивзодительность.

### *Этап 7: +++

   `Edit `→` Plugins` → поиск `Dynamic Wind` → поставить галку 
   `Project Setings` → `Nanite Foliage` → поставить галку 

   → `Restart`

3.  **Загрузи примеры Megaplants из Fab**
   Открой `Content Browser` → нажми на кнопку `Fab` вверху → в поиске введи `Megaplants` → добавь любой понравившийся пак в проект.

## Как сдавать

1. Сделайте скриншоты:
   - Вид поселения с окружающим лесом
   - Вид на лес издалека
   - Детальный вид на травы и камни
2. Напишите список использованных ассетов:
   - Названия деревьев (откуда из Fab)
   - Название травы/материала
   - Названия камней
3. Скриншот с FPS. Его можно влключить командой `stat fps`
4. Отзыв: было ли сложно выбирать оптимальные ассеты? Какие трудности столкнулись?

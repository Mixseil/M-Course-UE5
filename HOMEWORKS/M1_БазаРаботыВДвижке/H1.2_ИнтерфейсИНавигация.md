# Домашнее задание к Модулю: 1, Занятию: 2

[Конспект того занятия](LESSONS\M1_БазаРаботыВДвижке\L1.2_ИнтерфейсРедактора.md)
## Задание 1: Переставить панели интерфейса

1. Откройте свой проект UE5.
2. Переставьте панель **Details** так, чтобы она находилась слева от Viewport (вместо справа).
3. Создайте новую вкладку **Content Browser** в правой части экрана (рядом с Outliner).
4. **Дублируйте** Content Browser (зажмите Alt и перетащите вкладку) — должно быть две вкладки Content Browser'а в одной панели.
5. Вернитесь к стандартной раскладке: **Window → Reset Layout**.

**Что получится:** Вы поймёте, как гибко перестраивать интерфейс под свои нужды, и научитесь восстанавливать стандартную раскладку.

---

## Задание 2: Навигация в Viewport

1. Создайте новый уровень (File → New Level → Default / Blank) c любыми настройкамми.
2. Добавьте на сцену несколько простых объектов (Place Actors → Cube, Sphere, Cylinder). Разместите их в разных местах.
3. Используя только **ПКМ + WASD**, летайте между объектами.
4. Используя **Alt + ПКМ**, вращайте вид вокруг одного объекта.
5. Используя **Ctrl + колесо мыши**, переключайтесь между видами сверху, спереди, сбоку.
6. Используя **ЛКМ + Shift**, перемещайтесь в плоскости.

**Что получится:** Вы станете уверенно управлять камерой в редакторе и сможете быстро осматривать сцену с любого угла.

---

## Задание 3: Работа с Outliner и фильтрацией

1. На сцене с объектами из задания 2:
2. В **Outliner** создайте иерархию: перетащите 2-3 объекта в один объект (сделайте его родителем).
3. Сверните эту иерархию (нажимаю на треугольник слева).
4. Фильтруйте Outliner, вводя в поле поиска первую букву названия объекта (Cube, Sphere и т.д.) — посмотрите, как отфильтруются объекты.
5. Очистите фильтр (удалите текст из поля поиска).
6. **Дважды кликните** на одном из объектов в Outliner — камера должна на него сфокусироваться в Viewport.

**Что получится:** Вы освоитесь с работой Outliner'а и поймёте, как удобно искать и организовывать объекты на больших сценах.

---

## Задание 4: Переключение режимов отображения

1. На той же сцене:
2. Переключайтесь между режимами отображения (правый верхний угол Viewport):
   - **Lit** — нормальный режим.
   - **Unlit** — без освещения.
   - **Wireframe** — только сетка полигонов.
   - **Lighting Only** — только свет (объекты должны быть чёрные).
3. Добавьте **Point Light** (Window → Place Actors → Lights → Point Light) и посмотрите, как свет влияет в режиме **Lit**.
4. В режиме **Unlit** свет не должен влиять на отображение.
5. Поиграйтесь с другими режимами

**Что получится:** Вы поймёте разницу между режимами отображения и научитесь использовать их для отладки материалов и света.

## Задание 5: Консоль и команды

1. Откройте консоль клавишей **`~`** (тильда/ё).
2. Введите команду: `stat fps` — в верхнем левом углу должен появиться счётчик FPS.
3. Введите команду: `t.maxfps 30` — лимит FPS должен упасть до 30.
4. Введите команду: `t.maxfps 120` — верните лимит в нормальное значение.
5. Введите команду: `stat unit` — появится детальная информация о времени кадра (Game, Draw, GPU, Frame).
6. Закройте консоль ещё раз клавишей **`~`**.

## Задание 8: Сохранение собственной раскладки интерфейса

1. Переставьте панели интерфейса так, как вам удобнее всего работать.
2. Откройте **Window → Save Layout...** и дайте имя вашей раскладке (например, "MyLayout").
3. Установите эту раскладку как стандартную: **Window → Load Layout → MyLayout** (она будет загружаться при старте).
4. Откройте **Window → Reset Layout**, чтобы вернуться к стандартной раскладке.
5. Затем загрузите свою сохранённую раскладку обратно.

**Что получится:** Вы научитесь сохранять собственные конфигурации интерфейса и быстро переключаться между ними. Да и в целом запомните, что это можно делать, что круто

## Бонус: Горячие клавиши

Запомните эти клавиши и потестите:

- **Ctrl + Space** — открыть/закрыть Content Browser.
- **Ctrl + Tab** — переключение между вкладками в одной панели.
- **F11** — полноэкранный режим.
- **~** — консоль.
- **Ctrl + S** — сохранить текущий уровень.
- **Ctrl + W** — закрыть активное окно/вкладку.
